Privacy Policy — SCC BOM Warning Converter

Chrome Extension

Last updated: March 2026

This extension was developed by Jack as a passion project to support emergency management communications in Victoria, Australia. This project is not affiliated with, endorsed by, or operated by any entity, agency, or government.

Data collection

This extension does not collect, store, transmit, or share any personal data.

No analytics, tracking, telemetry, or advertising of any kind is included.

How it works

All data processed by this extension — including warning text, checklist states, publishing history, and settings — remains exclusively on your local device using Chrome's built-in chrome.storage.local API. No information is sent to any external server operated by the developer.

External services

The extension makes network requests to the following third-party services only:

  • Bureau of Meteorology (bom.gov.au) — to fetch public weather warning feeds and read warning detail pages
  • DataVic WFS (opendata.maps.vic.gov.au) — to fetch publicly available Emergency Management region boundaries for polygon matching

These requests are made directly from your browser and are subject to those organisations' own privacy policies.

Permissions

The extension requests the following Chrome permissions:

  • storage — to save warning data, checklist states, and settings locally on your device
  • alarms — to schedule periodic polling of the BOM warning feed
  • notifications — to alert you when new warnings are issued or existing warnings are updated or cancelled
  • activeTab / scripting — to read warning text from BOM web pages when you have one open
  • Host permissions (bom.gov.au, opendata.maps.vic.gov.au) — to fetch warning data and region boundaries from these public government APIs

Data retention

All data is stored locally and can be cleared at any time by uninstalling the extension or using the "Clear storage" function in the diagnostics panel. No data persists on any external server.

Contact

For questions about this extension or this privacy policy, contact the developer via the Chrome Web Store listing.
